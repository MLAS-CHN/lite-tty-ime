# lite-tty-ime（中文说明，v0.3.0）

这是一个运行在纯终端（TTY）里的拼音输入法程序。  
它会接管键盘输入，在终端内提供拼音候选、选词、词频学习、词库热重载等能力。

---

## 1. 项目现在有哪些功能（逐条）

### 1.1 终端运行模式
- Shell 包装模式：通过 PTY 启动你的 shell（默认 `/bin/bash`），输入法和 shell 同时工作。
- 底部状态栏：显示中英状态、当前拼音、候选信息等。
- 在 shell 模式下，输入法输出会直接写入 PTY，尽量不影响原有终端程序。

### 1.2 中英文输入切换
- 支持一个“切换键”（默认 `Ctrl+Space`）在中文输入和英文直通之间切换。
- 英文模式下，普通字符直接上屏；退格和回车按常规终端行为处理。
- 切换时会清理输入法临时状态，避免状态污染。

### 1.3 拼音输入与候选生成
- 支持字母输入（`a-z`）作为拼音。
- 支持手动分隔符 `'`。
- 内部会做多种拼音拆分策略：
  - 激进拆分（尽量多分）
  - 保守拆分（尽量少分）
  - 合并拆分（在合理处再合并）
- 候选来源包含：
  - 基础词库
  - 用户词库
  - 已学习短语库（`learned_phrases.txt`）的命中
- 支持“整串最佳候选”置顶（在可构造时）。

### 1.4 候选选择与翻页
- 数字键 `1..9`：选择当前页候选。
- `+` / `=`：下一页。
- `-`：上一页。
- 候选页大小可配置，范围 `1..9`（默认 5）。

### 1.5 输入动作策略（空格/回车可配）
- `space_action`、`enter_action` 都支持以下动作：
  - `select_first`：选第一个候选
  - `commit_raw`：直接上屏原始拼音
  - `send_line`：发送换行
  - `none`：不做处理

### 1.6 中文标点转换（条件触发）
- 当“当前没有拼音输入且没有候选”时，支持部分符号输出全角中文标点。
- 例如 `,` -> `，`，`.` -> `。`，`?` -> `？`，`(` -> `（` 等。
- 这样设计是为了避免与选词、编辑行为冲突。

### 1.7 光标与 ESC 序列处理
- 能识别常见 ESC 序列，避免输入法吞掉方向键/TUI 控制键。
- `Esc` 和方向键序列可透传给 shell 内程序。
- 在中文态中，支持 `Ctrl+Left` / `Ctrl+Right` 对拼音输入光标做左右移动（解析到对应序列时）。

### 1.8 词频统计与排序学习
- 每次选词后会更新词频文件：`<user_dict>.freq`。
- 候选排序会参考词频和最近使用时间（时间衰减+频次权重）。
- 结果是常用词会逐步靠前。

### 1.9 自动学习（写入用户词库/短语库）
- 连续选词后可自动判定是否写入：
  - 用户词库（`user_dict.txt`）
  - 学习短语库（`learned_phrases.txt`）
- 存储策略由规则函数控制（`learn_rules.*`），例如：
  - 连续输入结果是否算短语
  - 是否命中“包含已有词典词”的短语学习条件

### 1.10 词库热重载
- 会轮询词库文件修改时间，变更后自动重载，不需要重启程序。
- 热重载覆盖这些文件：
  - 基础词库
  - 用户词库
  - 词频文件
  - 已学习短语文件

### 1.11 启动命令与日志
- 支持 `--startup-cmd` 启动后自动执行一条 shell 命令。
- 支持 `--log-file` 记录运行日志（启动、热重载、写库、退出等）。
- 支持 `--debug` 打开调试输出。

## 2. 命令行参数（实际可用）

- `-h`, `--help`：显示帮助。
- `-V`, `--version`：显示版本。
- `-s`, `--startup-cmd "CMD"`：启动后执行命令。
- `--debug`：调试模式。
- `--dict PATH`：基础词库路径。
- `--user-dict PATH`：用户词库路径。
- `--config PATH`：配置文件路径（`key=value` 风格）。
- `--log-file PATH`：日志文件路径。
- `--toggle-key KEY`：中英切换键。
- `--exit-key KEY`：退出键。

> 说明：有些历史参数在代码里被明确标记为“已移除”，传入会报错退出。

## 4. 运行时会读写哪些文件

- 基础词库（只读）：`--dict` 指定路径。
- 用户词库（读写）：`--user-dict` 指定路径。
- 词频文件（读写）：`<user_dict>.freq`。
- 学习短语文件（读写）：`learned_phrases.txt`（位于用户词库同目录）。

当系统默认目录不可写时，会自动回退到用户目录（如 `$XDG_DATA_HOME` 或 `~/.local/share/...`）。

---

## 5. 键位速查

- `1..9`：选词
- `+` / `=` / `-`：翻页
- `'`：手动分隔拼音
- `Esc`、方向键：优先保证透传
- `Backspace`：删除拼音或回退已组成内容
- `Enter`/`Space`：按配置动作执行
